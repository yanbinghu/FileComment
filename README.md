# FileComment
vim plugin
## 功能
### 为.c,.h,.hpp,.sh文件自动添加文件头信息
```
/*================================================================
*   Copyright (C) 2017 . All rights reserved.
*   
*   文件名称：1.c
*   创 建 者：hyb
*   创建日期：2017年10月23日
*   描    述：
*
================================================================*
```
### 为函数添加自动添加注释信息
```
函 数 名：main
功能描述：
输入参数：char *argv[]
输出参数：
返 回 值：int

日    期：2017年10月23日
作    者：hyb
修改历史:新生成函数
```
### 添加函数修改信息


## 安装方法

### 安装Vundke

### 修改vimrc文件
添加内容
 ```
 Plugin 'https://github.com/yanbinghu/FileComment.git'
 ```
 
 ### 打开vim
 执行命令
 ```
 BundleInstall
 ```
